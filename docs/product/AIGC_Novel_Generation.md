# AIGC 小说创作助手 (AIGC Novel Writer) - 产品需求文档 (PRD)

## 1. 产品概述
**AIGC 小说创作助手** 是 SeekerAIStudio 的核心应用模块之一，旨在利用先进的大语言模型（LLM）能力，辅助创作者完成从灵感构思、大纲编排、角色设计到正文撰写、润色优化的全流程小说创作。

## 2. 目标用户
- **职业网文作者**: 需要辅助工具提高更新效率，优化卡文段落。
- **业余写作爱好者**: 有创意但缺乏长篇驾驭能力，需要 AI 辅助填充细节。
- **剧本/游戏文案策划**: 快速生成剧情分支和世界观设定。

## 3. 核心功能模块

### 3.1 创意与大纲 (Idea & Outline)
- **灵感风暴**: 用户输入关键词或简短描述，AI 提供多个题材、核心梗概供选择。
- **大纲生成**: 支持生成三级大纲（卷-章-节），并可对每一章的一句话梗概进行扩写。
- **剧情推演**: 针对大纲中的关键节点，分析剧情节奏（起承转合），提出优化建议。

### 3.2 角色工坊 (Character Workshop)
- **角色卡生成**: 自动生成角色姓名、性格、外貌、背景故事、口头禅等。
- **关系图谱**: 可视化展示角色之间的人物关系。
- **一致性检查**: 在写作过程中，AI 检查角色行为是否符合设定（Agent 监视）。

### 3.3 智能写作编辑器 (Smart Editor)
- **沉浸式写作**: 支持 Markdown 的富文本编辑器。
- **AI 续写**: 根据前文内容，提供多种走向的续写（如：战斗风格、情感风格、悬疑风格）。
- **指令式润色**: 选中一段文字，使用命令（如“改为悲伤的语气”、“增加环境描写”）进行重写。

### 3.4 知识库与世界观 (Lore & Knowledge)
- **世界观设定集**: 存储魔法体系、地理环境、历史背景等设定。
- **RAG 检索增强**: 写作时，AI 自动检索相关的世界观设定，避免设定冲突。

### 3.5 导出与发布
- 支持导出为 TXT, Markdown, EPUB, PDF 格式。

## 4. 用户流程 (User Flow)
1. **新建项目**: 输入书名、类型、预期字数。
2. **设定阶段**: 完善世界观、创建主要角色、生成粗纲。
3. **写作阶段**: 
   - 细化章节大纲。
   - 使用编辑器撰写正文，随时调用 AI 辅助（扩写、润色）。
   - 侧边栏实时查看角色卡和设定。
4. **审阅阶段**: AI 全文通读，检查逻辑漏洞和错别字。
5. **导出**: 生成最终文件。

## 5. 差异化优势
- **深度结合 Workflow**: 底层基于 SeekerAIStudio 的 Workflow 引擎，允许用户自定义“生成动作”的 Prompt 工作流。
- **本地隐私优先**: 支持通过 Ollama 链接本地模型，保障草稿隐私。
