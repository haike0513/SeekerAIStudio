# SeekerAIStudio UI/UX 与交互设计

## 1. 设计哲学
我们的设计理念以 **"简约中的力量 (Power in Simplicity)"** 为中心。我们旨在通过高端美学提供令人惊叹的第一印象，同时为复杂的逻辑构建保持一个整洁的环境。

### 1.1 美学支柱
- **高级深色模式**：深沉、丰富背景搭配高对比度强调色（主色调、玻璃拟态覆盖层）。
- **玻璃拟态 (Glassmorphism)**：面板和模态框上的微妙透明度和模糊效果，为工作区提供深度和上下文层次。
- **动态反馈**：交互元素对悬停、聚焦和激活状态做出响应，具有平滑的过渡效果。

## 2. 全局设计系统

### 2.1 排版 (Typography)
- **字体系列**: Inter / 无衬线系统栈。
- **层级/比例**: 标题 (H1-H3) 和正文之间有清晰的区别。代码片段和节点配置值使用等宽字体。

### 2.2 配色方案 (Color Palette)
- **背景**: 深岩板色 / 炭色 (Deep Slate / Charcoal)。
- **表面**: 卡片和面板使用较浅、略带透明的变体。
- **主色**: 充满活力的强调色（例如，电光蓝或霓虹紫），用于激活状态和主要操作。
- **状态**: 
  - 成功: 翡翠绿 (Emerald Green)
  - 错误: 玫瑰红 (Rose Red)
  - 警告: 琥珀色 (Amber)

### 2.3 组件库 (类 Shadcn)
- **按钮**: 圆角、扁平或幽灵变体。主要按钮具有渐变或发光效果。
- **输入框**: 极简边框，使用主色调的聚焦环。
- **模态/对话框**: 居中，背景模糊，缩放进入动画。
- **命令面板 (`Cmd+K`)**: 导航的核心——快速、键盘驱动的所有工具访问。

## 3. 交互模式

### 3.1 工作流编辑器 (SolidFlow)
- **画布导航**: 无限平移和缩放（滚动/捏合）。
- **节点交互**:
  - **拖动**: 带有吸附对齐的平滑移动。
  - **连接**: 从手柄点击并拖动以创建边。有效连接高亮显示；无效连接变暗。
  - **选择**: 点击单选，拖拽框选多选。
  - **上下文菜单**: 在画布或节点上右键单击以进行快速操作。
  
### 3.2 微交互 (Micro-interactions)
- **悬停效果**: 元素轻微上浮或发光。
- **加载状态**: 数据获取时的骨架屏；处理动作时的加载转轮。
- **Toast 提示**: 非侵入式通知（右下角），用于系统状态更新。

## 4. 无障碍性 (A11y)
- **键盘导航**: 完全支持无需鼠标即可导航图形和菜单。
- **对比度**: 确保文本可读性符合 WCAG AA 标准。
- **屏幕阅读器支持**: 所有交互控件上的 ARIA 标签。
